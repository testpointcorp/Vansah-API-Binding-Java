# API-Binding-Java

Vansah API 
------------

This repository contains several example bindings on how to access Vansah's API (v1).
You can learn more about Vansah's API here:

https://help.vansahapp.net/kb/vansah-api-reference-guide/

You may refer to the APIs here: http://apidoc.vansah.com/
------

Call endpoints as here:
			
add_test_Log
quick_test_update
dataset
session_variable
add_session_variable
test_script
email_reporting_logs
